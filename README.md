# Fallout-Inspired Interactive Terminal System

An interactive Fallout-themed terminal-style web application built using vanilla JavaScript. The project simulates a retro CLI interface with navigable screens, a custom command system, and an embedded radio player powered by the YouTube IFrame API.

---

## 🚀 Features

- Custom command interpreter for typed terminal input
- State-driven UI system for screen navigation
- Clickable UI options as an alternative navigation method
- Modular content architecture separating data from UI logic
- Embedded radio system using YouTube IFrame API
  - Playlist playback support
  - Custom controls (play, pause, next, previous, volume)
- Dynamic DOM manipulation without frameworks

---

## 🧠 Key Concepts

- State-based UI rendering
- Command parsing system (CLI-style interaction)
- Event-driven JavaScript architecture
- Separation of content and logic for scalability
- External API integration (YouTube IFrame API)

---

## 🛠 Tech Stack

- HTML5
- CSS3
